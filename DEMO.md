# Demo Shot List

1. Open sandboxed.sh Mission Control and show the existing product dashboard.
2. Create a workspace from `autonomous-transaction-safety-check`.
3. Start a mission: "Check this token, DApp URL, and transaction calldata with OKX security and produce a risk report."
4. Show the Library has one read-only `okx-security` skill selected.
5. Show the mission running in an isolated workspace, not on the host shell.
6. Close the laptop or disconnect from the dashboard while the agent continues.
7. Return to Mission Control and reopen the mission timeline.
8. Show the completed risk report with verdict, signals, and next action.
9. Point out the sandbox boundary: no keys, no signing, no swaps, no broadcasts.
10. End on the README one-liner: "the safe runtime for autonomous on-chain AI agents."
